## Research Findings

- Sentry DSN pattern (`https://key@host/project`) is worth stealing — simplifies SDK config to one env var
- Elixir SDK is 90% written — `CanaryTriage.ErrorReporter` is the template, just needs extraction + parameterization
- TypeScript SDK needs only `instrumentation.ts` + `global-error.tsx` integration — ~150 LOC, zero deps
- Cerberus already has `WebhookAlertSink` with HMAC — needs a thin adapter to match Canary ingest schema
- SSRF redirect bypass is a real security issue — Option A (disable redirects) is the recommended fix
- `auto_stop_machines = "stop"` is a critical misconfiguration for an uptime monitor

## Architecture Assessment

Codebase architecture is sound (deep modules, pure state machine, clean domain boundaries). Gaps are infrastructure, not design. No radical restructuring warranted.
