## Research Findings

- **Competitive position**: No other tool combines error tracking + uptime + SQLite + agent-first API. Closest: GlitchTip (PG+Redis), Uptime Kuma (no errors, Socket.IO API), ErrorTracker (library only).
- **Skills vs MCP**: For single-user Claude Code usage, SKILL.md is strictly better. MCP adds 500+ LOC runtime for marginal benefit.
- **Dashboard pattern**: ErrorTracker proves 2-3 LiveViews + 5 components is sufficient. Gatus uptime bar (flexbox colored spans) is the cleanest visualization.
- **Sentry wire protocol**: Rejected. Adds protocol translation complexity for mass-market adoption Canary doesn't need.
