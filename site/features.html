Features

Canary is one Rust service, one SQLite file, and a set of agent-facing surfaces for health, incidents, timelines, and queryable production context.

job See every monitored service's health, open incidents, and error rate on one dashboard. proof Dashboard overview and incident drill-down, screenshots below.
shape TTL monitors, HTTP target probes, and a signed error-ingest API feed one incident correlation engine. trust Self-hosted: your SQLite file and your own Docker host, no vendor holding your production data.
delivery bin/canary doctor proves a fresh install end to end before you trust it with real traffic. fit No Fly account required; plain docker compose up works too.
